N상승 반등타기
================

코인 5분봉에 적용한 N 패턴
-------------------------
>주가는 상승-하락-상승의 단계를 거쳐서 
>계단식으로 상승하게 되는데
>두번째 상승이 첫번째 상승보다 높은 지점에서 반등을 시도할때
>매수를 하면 두번째 상승구간에서 이익을 취할 수 있다.
